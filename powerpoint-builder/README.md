# powerpoint-builder

Build a polished, editable **PowerPoint (.pptx)** from content you already have. You bring the words;
the skill designs and lays out the deck. It does **not** write the presentation for you or invent facts.

Pick one of three looks:

- **Your organization's brand** (pulls from a brand skill if you have one installed, or asks you for
  your colors, fonts, and logo rules)
- **Match a template you already use**
- **A new custom style**

Works in Chat, and in Cowork or Claude Code where it can save the `.pptx` into your workspace.

## Install

In Claude Code:

```
/plugin marketplace add Dwell-Community-Church/dwell-plugins
/plugin install powerpoint-builder@dwell-plugins
```

Then say *"turn this outline into a PowerPoint."*
