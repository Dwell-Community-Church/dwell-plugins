# staff-onboarding

Personalize Claude for a church or ministry staff member. It runs a short, friendly interview (leaning
on what Claude already remembers about them), then produces a **Custom Instructions block** they paste
into their profile so every chat starts on the same page. In **Cowork** or **Claude Code** it also
scaffolds a workspace and starter context files.

Built to be easy and quick: it offers tappable options on every question, confirms rather than assumes,
and doesn't re-interview someone who's already set up.

## Customize it for your church

Open the skill and edit the **"Org background"** block near the top: list anything every staff member
shares (your church management system and version, any brand or documentation skills that already reach
staff) so the interview stays about the *person*. Delete the block if you have nothing to add.

## Install

In Claude Code:

```
/plugin marketplace add Dwell-Community-Church/dwell-plugins
/plugin install staff-onboarding@dwell-plugins
```

Then say *"help me personalize Claude"* or *"set me up."*
